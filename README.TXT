Sprint 4

Nesta sprint foi feito a intengração do API https://viacep.com.br nas páginas de cadastro do cliente e no cadastro de comércio. Obtendo o valor do campo "CEP" e preenchendo, automaticamente, os campos de logradouro, bairro, cidade e estado. Além disso, foi feito o bloqueio de usuários menores de idade, pendência falada durante as apresentações da última semana.

E para a próxima sprint, pretendemos adicionar um banco de dados para múltiplos usuários, e um para múltiplos estabelecimentos. Para podermos assim, em seguida, adicionarmos funcionalidades como filtragem, amigos de usuários, estabelecimentos favoritos, eventos disponíveis, entre outros.
